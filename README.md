# e-plantShopping

**Project Name:** e-plantShopping

## Overview
e-plantShopping is a React-based web application that allows users to browse and purchase plants online from Paradise Nursery.  
The application provides an intuitive and user-friendly shopping experience, enabling users to explore different plant categories, add plants to a shopping cart, and manage their selections before checkout.

## Application Description
Paradise Nursery is an online plant shop that offers a variety of houseplants such as aromatic, medicinal, and decorative plants.  
This application demonstrates core front-end development concepts using React and Redux, including component-based architecture, state management, routing, and dynamic rendering of data.

Users can:
- Start from a visually appealing landing page
- Browse plants grouped by categories
- View plant images, names, prices, and descriptions
- Add plants to a shopping cart
- Increase or decrease plant quantities
- Remove plants from the cart
- View the total number of items and total cost in real time

## Features
- Landing page with an interactive **Get Started** button
- Plants catalog page displaying categorized plant cards
- Shopping cart with quantity controls and delete option
- Dynamic cart icon showing total items
- Responsive navigation bar with links to Home, Plants, and Cart

## Technologies Used
- React (Functional Components & Hooks)
- Redux Toolkit for state management
- React Router for navigation
- Vite for project setup and build
- GitHub Pages for deployment
