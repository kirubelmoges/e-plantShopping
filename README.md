# e-plantShopping

**Project Name:** e-plantShopping

## Overview
e-plantShopping is a React-based web application that allows users to browse and purchase plants online. 
The application features a clean landing page, a plants catalog page, and a shopping cart for users to manage their selected items.

### Features
- Landing page with an interactive "Get Started" button
- Plants page displaying plant details
- Shopping cart functionality to manage items
- Responsive navigation bar with links to Home, Plants, and Cart

## Repository
This repository contains all the source code, assets, and documentation required to run the e-plantShopping application.
